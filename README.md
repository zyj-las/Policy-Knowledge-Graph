# Policy-Knowledge-Graph
本仓库为政策知识图谱项目的代码仓库，不包含任何敏感数据。主要用于抽取多维细粒度政策知识要素，包括政策主体、条文、引用、政策目标、政策工具、政策措施以及作用群体、物体、主体、行为等核心要素内容。
# 环境
-python3.12.1  
-推荐使用虚拟环境 conda
# 文件夹说明
data：过程数据集；policy_txt:示例政策文本  
modles：模型（注意：由于GitHub文件大小限制，训练模型不包括在此存储库中。）
# 程序功能说明
## 基于索引匹配的政策要素内容抽取
1-1政策主体清洗和分类程序  
1-2政策发布匹配程序
## 基于全文解析的政策要素内容抽取
2-1政策条文批量分割程序（标题-章节-段落）  
2-2 政策条文自动筛选程序（根据对应标准）  
3-1基于政策全文本的政策引用抽取  
3-2政策引用类型匹配程序
## 基于实体识别的政策要素内容抽取
4-作用客体_词典模型  
包含作用群体、物体、主体
## 基于文本分类的政策要素内容抽取
### BERT 系列模型
5-1-1 政策目标、工具、措施、作用行为：BERT 模型训练  
5-1-2 政策目标、工具、措施、作用行为：BERT 模型预测
### DistilBERT 系列
5-2-1 政策目标、工具、措施、作用行为：DistilBERT 模型训练  
5-2-2 政策目标、工具、措施、作用行为：DistilBERT 模型预测
### TinyBERT 系列
5-3-1 政策目标、工具、措施、作用行为：TinyBERT 模型训练  
5-3-2 政策目标、工具、措施、作用行为：TinyBERT 模型预测
